# 中国象棋

#### 介绍

基于HTML、CSS、原生 JavaScript制作的中国传统游戏——中国象棋

#### 项目结构

1.  使用了ECMAScript ES6 语法 
2.  插件使用了babel-cli：V6.26.0"、babel-preset-es2015: V6.24.1
3.  ./src/index.js是入口文件，编译后的JS文件为./dist/index.js
4.  图标、CSS、HTML文件在public目录下

#### 使用说明

1.  实现了中国象棋的所有规则
2.  非AI对战，仅能用鼠标指挥两家互杀
3.  一局结束后刷新页面重新开始
4.  内含控制台中文输出
